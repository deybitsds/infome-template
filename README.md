# Template Informe

Template recuperado de: [Este repositorio](https://github.com/Template-Latex/Template-Informe/) 


## Manual de uso

Usa el script de compilación

### Para main.tex
```bash
make
```

### Para otro archivo
```bash
make <file_name.tex>
```
